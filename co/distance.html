Applications

Des distances

Supposons qu'on habite Le Mans et que l'on dispose de 20 litres dans notre réservoir. Notre véhicule consomme 6 litres au 100 kms :

  • Chercher en utilisant google maps la distance entre Le mans et les villes suivantes : Rennes, Nantes, Saint Nazaire, La Rochelle, Paris, Nancy, Amiens, Lille ;

  • Reporter ces villes et distances (la plus courte) dans un tableau (utiliser le format personnalisé : 000" kms");

  • Écrire une fonction qui inscrit Vrai ou Faux suivant s'il est possible de se rendre dans chacune de ces villes avec la quantité d'essence dans le réservoir ;

  • Utiliser ensuite une fonction SI qui permettra d'inscrire : la quantité d'essence restante s'il est possible d'atteindre la ville, "Impossible" sinon.

Mon garage

On possède un garage et on se demande s'il est possible d'empiler un certain nombre d'objets dans ce garage pour les stocker :

  • En colonne A, on inscrit une liste d'objets, en colonne B leur hauteur ;

  • En cellule C1, on entre la hauteur du garage ;

  • En cellule C2, entrer une formule utilisant la fonction somme pour définir s'il est possible d'empiler ces objets dans le garage, cette formule affiche Vrai ou Faux ;

  • Utiliser la fonction Si pour qu'en fonction de cette formule, on affiche soit "Possible", soit la hauteur en trop de façon à pouvoir décider quel objet enlever.
