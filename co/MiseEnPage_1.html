Mise en page d'une feuille

Mise en page d'une feuille

Nous allons présenter complètement un tableau de données [xls] issu de la base de données eurostats.

Nous allons effectuer la présentation de ce tableau en ajoutant une feuille[1] et en présentant cette feuille.

Mise en forme de cellules

Mise en forme de cellules

Mettre en forme du texte et des caractères individuels.

Pour faire ressortir du texte, vous pouvez mettre en forme tout le texte d'une cellule ou certains caractères sélectionnés. Sélectionnez les caractères que vous voulez mettre en forme, puis cliquez sur un bouton dans la barre d'outils Mise en forme ou bien dans le menu Format.

Nommer des zones de la feuille

Attribuer un nom aux cellules d'un classeur

Vous pouvez utiliser les étiquettes de colonnes et de lignes dans une feuille de calcul[1] pour faire référence aux cellules situées dans ces colonnes et lignes. Vous pouvez également créer des noms descriptifs pour représenter les cellules, les plages de cellules, les formules ou les valeurs de constante. Les étiquettes peuvent être utilisées dans les formules qui font référence aux données dans la même feuille de calcul ; si vous souhaitez représenter une plage sur une autre feuille de calcul, utilisez un nom.

Pour définir un nom, il faut :

1. Sélectionner la zone;

2. [Formule[définir un nom;

3. Donner le nom à la zone.

Un "Gestionnaire de noms" est à votre disposition dans l'onglet "Formules".

Atteindre une zone nommée

Retrouver une zone

Une plage de cellules identifiée par un nom peut être retrouvée :

- [Accueil[édition[Rechercher[Atteindre ;

- En allant dans la zone nom ;

- En utilisant le Gestionnaire de noms.

Sauvegarde du document

Méthode : Choisir le nom et le type du document à sauvegarder

Pour sauvegarder son document, il faut :

  1. Aller dans le menu fichier ;

  2. Choisir enregistrer sous ;

  3. Entrer le nom du classeur dans la case Nom de Fichier ;

  4. Choisir le type de fichier dans la liste déroulante du type.

Une fois ces opérations faites, il faut cliquer régulièrement sur l'icône pour enregistrer (ou appuyer sur Ctrl S).

Notes

  1. Feuille de calcul

    Un classeur contient une ou plusieurs feuilles de calcul.

    Vous pouvez entrer ou modifier des données, des formules dans une feuille de calcul et exécuter des calculs en fonction des données de la feuile de calcul ou d'une autre feuille du même classeur (nous ne verrons pas les données issus d'autres classeurs).

    Vous pouvez vous déplacer d'une feuille de calcul en cliquant sur les onglets de feuille situés en bas de la fenêtre du classeur.

    Vous pouvez coder par couleur les onglets de votre feuille pour les reconnaître plus facilement en utilisant le menu contextuel sur l'onglet feuille.

    Vous pouvez créer une nouvelle feuille soit par [insertion[feuille ou bien avec le menu contextuel sur l'onglet feuille.

    L'onglet correspondant à la feuille active est souligné dans la couleur choisie; les onglets pour feuilles inactives sont totalement colorées.
