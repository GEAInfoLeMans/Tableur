Retour sur la validation de données

Simulation :

Dans cet exemple dans la case B4 , seul les noms définis en A8:A10 pourront être saisi à l'aide d'une liste déroulante qui sera placée sur la cellule B2.

La liste sera dans la cellule B4 et les noms Durand, Dupont et Smith apparaîtront.

Attention : Contrainte

Les données qui sont dans la liste déroulante sont obligatoirement sur la même feuille lorsque vous utilisez la souris.

Pour utiliser une liste dans une autre feuille vous devez nommer la liste et mettre le nom manuellement.
