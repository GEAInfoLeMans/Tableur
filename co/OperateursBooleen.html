Opérateurs Booléens

Excel peut utiliser des données de type numérique, de type texte ou de type booléen. Une donnée de type booléen ne peut avoir que deux valeurs: VRAI ou FAUX.

Par exemple :

  • 14>15 est FAUX ;

  • 15>=14 est VRAI (>= signifie supérieur ou égal à) ;

  • 14<>15 est VRAI (<> signifie différent de).

Dans une formule, une donnée de type texte est placée "entre guillemets". Pour que dans une cellule, un nombre soit considéré comme de type texte, il doit être saisi précédé d'une apostrophe.

Syntaxe : Traduction en excel des comparaisons précédentes

  • =14>15 ;

  • =15>=14 ;

  • =14<>15.

Opérateur ET

L'opérateur ET est utilisé lorsque toutes les conditions doivent être réalisées simultanément (vraies simultanément). Si toutes les conditions sont vraies alors le résultat sera vrai.

Exemple

  • 15>14 ET 17>0 est une condition vraie. ;

  • 15>17 ET 17>0 est une condition fausse.

Syntaxe : Traduction en excel des opérateurs ET précédents

  • =ET(15>14 ;17>0) ;

  • =ET(15>17 ;17>0).

Opérateur OU

L'opérateur OU est utilisé lorsque une des conditions doit être réalisée . Si une seule des conditions est vraie alors le résultat sera vrai.

Exemple

  • 15>14 OU 17>0 est une condition vraie. ;

  • 15>17 OU 17>0 est une condition vraie. ;

  • 15>18 OU 17>21 est une condition fausse.

Syntaxe : Traduction en excel des opérateurs OU précédents

  • =OU(15>14 ;17>0) ;

  • =OU(15>17 ;17>0) ;

  • =OU(15>18 ;17>21).
