Fonctions financières

Introduction

Définition :

Pour étudier un placement ou un prêt avec Excel, il faut connaître les notations suivantes :

  • taux = taux d'intérêt (annuel) appliqué au prêt ou au placement ;

    npm = Nombre total de versements dans le cadre du prêt, sinon durée du placement ;

    vpm = Montant du remboursement de chaque période (ex. mensualités) dans le cadre d'un prêt ;

    va = Valeur actuelle (montant du placement ou montant du prêt) ;

    vc= Valeur capitalisée, c'est à dire la valeur future dans le cadre d'un placement.

Ainsi, dans le cadre d'un prêt, on s'intéresse à :

  • taux = taux d'intérêt (annuel) appliqué au prêt ;

    npm = Nombre total de versements de remboursement ;

    vpm = Montant du remboursement de chaque période ;

    va = Valeur actuelle (montant du prêt).

Et dans le cadre d'un placement :

  • taux = taux d'intérêt (annuel) appliqué au placement ;

    npm = la durée du placement ;

    va = Valeur actuelle (montant du placement) ;

    vc= Valeur capitalisée, c'est à dire la valeur future du capital placé.

Excel fournit pour chacune de ces valeurs une fonction associée. Ainsi, il existe des fonctions VC, NPM, VPM, VA et TAUX dans Excel qui permettent de calculer l'une de ces valeurs en fonction des autres.

Par exemple, la documentation Office de la fonction NPM (Lien) nous dit que la fonction s'utilise ainsi :

NPM(taux,vpm,va,[vc],[type])

Les termes entre crochet sont facultatifs.

Toutes les autres fonctions sont également décrites dans le documentation Office (il suffit de les googliser).

Exemple : NPM

Nous savons maintenant comment utiliser la fonction NPM. Tentons de l'utiliser sur un exemple.

Nous souhaitons placer 10 000€ (VA) à la banque pour atteindre 11 000€ (VC) via ce placement. La banque nous propose un placement à 2,50% (TAUX).

Combien de temps nous faudra-t-il pour atteindre les 11 000€ avec ce taux ?

NPM(taux ; vpm ; va ; vc) = NPM(0,025 ; 0 ; -10 000 ; 11 000)

La Valeur Actuelle est négative dans ce cas car il s'agit d'un placement.

En entrant la formule ci-dessous dans une cellule, vous obtenez

3,85986616262265 -> Il faudra un peu moins de 4 ans pour atteindre 11 000€.

Exemple : VPM

Autre exemple, cette fois-ci avec la fonction VPM :

Nous empruntons 10 000€ (VA) à un taux de 2,50% (TAUX) sur 5 ans (NPM).

À combien se montent les mensualités (VPM) pour le remboursement du prêt ?

Nous utilisons la fonction :

VPM(taux, npm, va, [vc], [type])

VPM(0,025 /12 ; 5*12 ; 10000)

Pour obtenir les mensualités (mois), nous avons divisé le taux annuel par 12, et multiplié le nombre d'année par 12.

Le résultat donne 864,12 €.
