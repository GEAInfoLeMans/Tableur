Validation de données

Méthode : Accès

Pour valider les données d'une zone il faut :

  1. Sélectionner les données où seront saisies les données ;

  2. Cliquer sur l'icône validation de données dans le groupe outils de données du ruban données.

Contrôler les données

Le contrôle peut porter sur le type de la donnée saisie (entier, décimal,...), sur sa position par rapport à des valeurs limites, sur son appartenance à une liste prédéfinie de valeurs ou être plus personnalisé: il porte alors sur la réalisation d'une expression booléenne.

Définition : Validation de données

La validation de données permet de vérifier la cohérence d'une donnée par rapport à un intervalle, une liste de données ou bien une expression booléennes.

Des messages d'aide à la saisie ou d'alerte en cas d'erreur peuvent être affichés.

Complément : Utilisation d'expression booléenne

On peut autoriser la saisie de données dont le résultat d'une formule logique donne VRAI.
