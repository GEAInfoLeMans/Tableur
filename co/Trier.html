Tri d'un tableau de données

Méthode : Comment trier

Il existe un icône sous Excel pour trier. Cet icône permet de visualiser ce que l'on va faire lors du tri et d'accéder aux différentes options. Une autre possibilité et d'utiliser le menu contextuel après avoir sélectionner le tableau à trier.

Simulation : Trier le tableau par croissant des notes obtenus en Français et par ordre alphabétique en cas d'égalité.

Pour trier le tableau, il faut :

  1. sélectionner l'ensemble du tableau ;

  2. [Données[Trier]] :

  3. Sélectionner la colonne C dans la case "trier par" ;

  4. Sélectionner la colonne A dans la case "puis par".

Un exemple de tri ou bien :

Remarque : Option du tri

  • Lorsque les colonnes sont nommées à la place de colonne A apparaît le nom de la colonne. Vous pouvez forcer une ligne de titre en cliquant dans la case correspondante ;

  • S'il faut trier une ligne, il faut aller dans les options sélectionner une orientation "de la gauche vers la droite".

Attention :

Il faut sélectionner l'ensemble des données, toutes les lignes et les colonnes, pour trier. Si une partie des données n'est pas sélectionnée alors les données seront mélangées...
