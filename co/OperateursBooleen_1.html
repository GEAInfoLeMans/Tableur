Opérateurs Booléens

Excel peut utiliser des données de type numérique, de type texte ou de type booléen. Une donnée de type booléen ne peut avoir que deux valeurs: VRAI ou FAUX.

Par exemple :

  • 14>15 est FAUX ;

  • 15>=14 est VRAI (>= signifie supérieur ou égal à) ;

  • 14<>15 est VRAI (<> signifie différent de).

Dans une formule, une donnée de type texte est placée "entre guillemets". Pour que dans une cellule, un nombre soit considéré comme de type texte, il doit être saisi précédé d'une apostrophe.

Syntaxe : Traduction en excel des comparaisons précédentes

  • =14>15 ;

  • =15>=14 ;

  • =14<>15.
