Fonctions financières

La fonction VPM()

La fonction VPM() permet de calculer la valeur de chaque montant périodique requis pour rembourser une dette à un taux d'intérêt constant.

=VPM(taux ; nombre_de_paiements ; valeur_actuelle ; [valeur_future] ; [type-paiement] )

  • TAUX : Taux périodique (i) ;

  • NPM :Nombre de versements (n) ;

  • VA :Valeur actuelle des versements (PV) ;

  • VC :Valeur acquise des versements (FV) ;

  • TYPE : 0 si annuité de fin de période et 1 si annuité de début de période.

La fonction INTPER()

La fonction INTPERM() permet de calculer le montant des intérêts dus pour une période donnée dans le cadre d'un emprunt à annuités constantes.

INTPER(taux, pér, npm, va, [vc], [type])

  • TAUX : Obligatoire. Représente le taux d'intérêt par période ;

  • PER :Obligatoire. Représente la période pour laquelle vous souhaitez calculer les intérêts. La valeur spécifiée doit être comprise entre 1 et npm ;

  • NPM : Obligatoire. Représente le nombre total de périodes de remboursement au cours de l'opération ;

  • VA : Obligatoire. Représente la valeur actuelle, c'est-à-dire la valeur, à la date d'aujourd'hui, d'une série de versements futurs ;

  • VC : Facultatif. Représente la valeur capitalisée, c'est-à-dire le montant que vous souhaitez obtenir après le dernier paiement. Si vc est omis, la valeur par défaut est 0 (par exemple, la valeur capitalisée d'un emprunt est égale à 0) ;

  • TYPE : Facultatif. Peut prendre les valeurs 0 ou 1, et indique l'échéance des paiements. Si vous ne spécifiez pas l'argument type, sa valeur par défaut est 0.

Méthode : Réaliser un tableau de remboursement d'emprunt à annuités constantes

Tableau de remboursement d'emprunt à annuités constantes  [xls]:

1/ Dans la colonne "Période", organiser l'affichage automatique du numéro de l'année ;

2/ Dans la colone "Montant restant dû au début", reporter le montant du capital restant dû au début de la période ;

3/ A l'aide de la fonction VPM, calculer le montant de l'annuité constante de la période ;

4/ A l'aide de la fonction NPM, calculer le montant de l'intérêt dû de la période ;

5/ Calculer le montant du capital remboursé de la période ;

6/ Calculer le montant du capital restant dû à la fin de la période ;

7/ Appliquer un format "Personnalisé" adéquat pour chaque cellule ;

8/ Recopier les formules vers le bas pour obtenir les valeurs sur cinq ans ;

9/ Vérifier les résultats à l'aide de l'application 3 du chapitre 4 (cas de l'annuité constante) de l'excellent cours de Mathématiques financières de 1ère année de Madame DERROISNE ;

10/ Recopier les formules vers le bas pour envisager un remboursement sur dix ans.
